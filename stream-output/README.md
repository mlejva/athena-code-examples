# How to start the demo

## 1. Start the server
1. Go to `./server` directory
2. Copy `.env.example` to `.env` and set your E2B API key
3. Install dependencies
```sh
poetry install
```
4. Start the server
```sh
poetry run start
```

## 2. Start the webapp
1. Go to `./webapp` directory
2. Install dependencies
```sh
npm i
```
3. Start the webapp
```sh
npm run dev
```
